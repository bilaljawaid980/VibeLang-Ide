    Program Basics

    • End every statement with semicolon.
    • Declare variable before using it.
    • Use print for output.
    declare variable marks = 75;
print marks;

    Control Flow

    if marks is greater than 50 then
    print "pass";
else
    print "fail";
end if;
    declare variable i = 1;
while i is less than or equal to 3 do
    print i;
    i = i + 1;
end while;

    Supported Operators

    • Math: +, -, *, /
    • Compare: is less than, is greater than
    • Compare: is equal to, is not equal to
    • Compare: is less than or equal to
    declare variable total;
total = 10 + 5 * 2;
print total;

    Common Mistakes

    • Missing semicolon at line end.
    • Using undeclared variable.
    • Forgetting end if; or end while;
    • Invalid English comparison phrase.
