🚀 Getting Started

Your First Program

Let's start with the classic "Hello World" program:

hello.vibe
print "Hello, VibeLang!";

Run this in the IDE and you'll see the output in the Terminal panel at the bottom. That's it! You've written your first VibeLang program.

How VibeLang Works

  1. Write: Type your code in the editor
  2. Compile: Press the Run button or use the compile endpoint
  3. See Results: View output in the Terminal panel
  4. Debug: Check errors and TAC (Three-Address Code) for learning
💡 Tip: Use the built-in documentation tabs in the IDE sidebar to quickly reference syntax while coding.

📝 Syntax Basics

Variables and Declaration

Variables store data that your program uses. Declare a variable with the var keyword:

Variable Declaration
var age = 25; var name = "Bilal"; var score = 95;

Variable names must start with a letter and can contain letters, numbers, and underscores.

Assignment

Change a variable's value with the assignment operator =:

Variable Assignment
var counter = 0; counter = 10; counter = counter + 5;

Print Statements

Display output using the print statement:

Using Print
print "Welcome to VibeLang"; var age = 25; print age;
Output:
Welcome to VibeLang
25

Comments

Comments start with -- and help explain your code:

Adding Comments
-- This is a comment var x = 10; -- Initialize x print x;
⚠️ Important: All statements must end with a semicolon ;

🎯 Data Types

Integers

Whole numbers (positive, negative, or zero):

Integer Examples
var age = 25; var temperature = -5; var count = 0;

Strings

Text enclosed in double quotes:

String Examples
var greeting = "Hello, World!"; var name = "Alice"; print "VibeLang is awesome";

String Concatenation

Combine strings using the + operator:

Concatenating Strings
var first = "Hello"; var last = "World"; var message = first + " " + last; print message;
Output:
Hello World

Arithmetic Operations

Perform calculations with integers:

Math Operations
var a = 10; var b = 3; print a + b; -- 13 print a - b; -- 7 print a * b; -- 30 print a / b; -- 3 print a % b; -- 1

Operators: + (add), - (subtract), * (multiply), / (divide), % (modulo/remainder)

🎮 Control Flow

If Statement

Execute code only if a condition is true:

If Statement
var age = 18; if age >= 18 then print "You are an adult"; end if;

If-Else Statement

Execute one block if true, another if false:

If-Else Statement
var score = 45; if score >= 50 then print "Pass"; else print "Fail"; end if;

While Loop

Repeat code while a condition is true:

While Loop
var count = 1; while count <= 5 do print count; count = count + 1; end while;
Output:
1
2
3
4
5

Comparison Operators

Used in conditions:

  • == - Equal to
  • != - Not equal to
  • < - Less than
  • > - Greater than
  • <= - Less than or equal to
  • >= - Greater than or equal to

⚙️ Functions

Defining a Function

Functions let you reuse code. Define with function:

Function Definition
function greet() print "Hello!"; end function;

Calling a Function

Execute a function by calling its name with parentheses:

Calling Functions
function greet() print "Hello!"; end function; greet(); -- Call the function greet(); -- Call it again

Functions with Parameters

Pass data to functions:

Function Parameters
function greetPerson(name) print "Hello, " + name; end function; greetPerson("Alice"); greetPerson("Bob");
Output:
Hello, Alice
Hello, Bob

Functions with Return Values

Functions can return a value using return:

Return Statement
function add(a, b) return a + b; end function; var result = add(5, 3); print result; -- 8

Function Scope

Variables inside a function are local to that function:

Variable Scope
var x = 10; -- Global x function example() var x = 20; -- Local x (only in function) print x; end function; example(); -- Prints 20 print x; -- Prints 10
💡 Tip: Use functions to organize your code and avoid repeating the same code multiple times.

📌 Complete Examples

Example 1: Temperature Converter

Convert Fahrenheit to Celsius:

temp_converter.vibe
function fahrenheitToCelsius(fahrenheit) var celsius = (fahrenheit - 32) * 5 / 9; return celsius; end function; var temp_f = 98; var temp_c = fahrenheitToCelsius(temp_f); print "98°F = " + temp_c + "°C";

Example 2: Factorial Calculation

Calculate factorial of a number:

factorial.vibe
function factorial(n) if n <= 1 then return 1; end if; return n * factorial(n - 1); end function; print factorial(5);

Example 3: Sum of Numbers

Sum numbers from 1 to N:

sum.vibe
function sumTo(n) var sum = 0; var i = 1; while i <= n do sum = sum + i; i = i + 1; end while; return sum; end function; print sumTo(10);

❌ Common Mistakes

Missing Semicolons

❌ Wrong:
print "Hello"
✅ Correct:
print "Hello";

Variable Not Declared

❌ Wrong:
age = 25; -- Missing 'var' keyword
✅ Correct:
var age = 25;

Mismatched If-End Blocks

❌ Wrong:
if x > 5 then print x; end; -- Missing 'if' keyword
✅ Correct:
if x > 5 then print x; end if;

Function Called Before Definition

❌ Wrong:
add(2, 3); -- Function not defined yet! function add(a, b) return a + b; end function;
✅ Correct:
function add(a, b) return a + b; end function; add(2, 3); -- Now defined!

📋 Syntax Reference & Cheat Sheet

Keywords

Variable & Data

var name = value; print value; "string" 123

Conditionals

if condition then ... end if; if condition then ... else ... end if;

Loops

while condition do ... end while;

Functions

function name(params) ... return value; end function;

Operators

Arithmetic

+ add - subtract * multiply / divide % modulo

Comparison

== equal != not equal < less than > greater than <= less/equal >= greater/equal

String

+ concatenate "text" literal

Assignment

= assign ; end statement

Quick Reference

  • Statement terminator: All statements end with ;
  • Comments: Start with --
  • String quotes: Use double quotes "text"
  • Then/Do: Required after conditions and while
  • End blocks: end if;, end while;, end function;
